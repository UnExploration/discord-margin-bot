
Project - Old School Runescape Discord Bot. 
A discord bot that keeps track of margins and price info in the OSRS market.
Uses a rich embed that continually updates with price info taken from the OSRSWiki/Runelite API


If you want to use it for yourself. Populate an env file with your discord bot Secret Key/Token as well as a user agent Header and finally change the ID of the channel.
